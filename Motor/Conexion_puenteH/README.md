## Elementos:

* 2 optocopladores tipo 4N35
* 2 transistores irlb8721 canal n (los que se colocan abajo)
* 2 transistores fqp27p06 canal p (los que se colocan arriba)
* Motor reciclado de impresora hp dekjet 845
* Placa de arduino UNO
* Resistencias (330 ohmios, 10000 ohmios??)
* Diodo IN4001?
* Fuente de tensión de corriente continua de 12V?
